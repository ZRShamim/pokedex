## Code Flow:
For this project I used Getx for state management and also navigation. Followed MVC patteren in this project. All the pages are in view folder.

```
└── lib/
    ├── api module/
    │   
    ├── constant/
    │   
    ├── controllers/
    │   
    └── model/
    │   
    └── view/
        │
        └── common widgets/
        │
        └── pages/
```
